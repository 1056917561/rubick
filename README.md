 <h1>Rubick2</h1>

基于 electron 的开源工具箱，自由集成丰富插件，打造极致的桌面端效能工具。Rubick(拉比克) 是 dota 里面的英雄之一，其核心技能是插件化使用其他英雄的技能，用完即走。非常符合本工具的设计理念，所以取名 Rubick。

## 支持能力
- [x] 支持远程下载安装插件，支持插件开发者模式
- [x] 支持系统插件，只要 rubick 在运行，即可使用，不需要搜索呼起
- [x] 支持全局快捷键设置
- [x] 支持搜索本地已安装 app 或 偏好设置
- [ ] 支持 Windows
- [ ] 支持 Linux
